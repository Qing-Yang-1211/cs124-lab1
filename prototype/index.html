To-Do List Walkthrough

The empty list

This demonstrates an empty to-do list with no items. The input field at the bottom allows the user to enter the name of an item and click on the "plus" button to add a new item. All the finished items will be hidden when the user clicks on the "Hide Finished" button at the top, and all the finished items will be deleted when the user clicks on the "Delete Finished" button at the top.

After creating a new item in empty list

This shows the page after the user entered "Buy new John Grisham book" in the input field at the bottom and clicked the "plus" button. A new item named "Buy new John Grisham book" has appeared as the first row and the only row on the to-do list. This item will be marked as complete if the user checks the checkbox associated with it. It will be removed if the user clicks on the "trash" sign, and the user can rename this item by clicking on the "pencil" sign. Currently, the to-do list is not empty and we will create a new item in this non-empty list.

After creating a new item in non-empty list

This page exists after the user creates a new item named "Eat Lunch" in the non-empty list by entering "Eat Lunch" in the input field at the bottom and clicking the "plus" button besides it.

Before marking "Call Mom" completed

This shows the to-do list with three items, including the "Call Mom" item. In the next step, we are going to mark the "Call Mom" item as completed.

After marking "Call Mom" completed

This is the page after the user marked the "Call Mom" item as completed. The user marked the item by checking the checkbox before the "Call Mom" text. The background color of the row containing this item changed to a light green color, indicating that this item is completed.

Before renaming an item

This is the page after the user created a "Text John" item. Notice that all the unfinished items are shown above all the finished items and items are in the same order as they were created. In the next step, we are going to rename the "Text John" item to "Text John about bank statements".

After renaming an item

This is the page after the user has renamed the "Text John" item to "Text John about bank statements" by clicking on the "pencil" sign, deleting the old name and entering the new name.

Show only uncompleted items

This is the page after the user clicked the "Hide Finished" button. The "Hide Finished" button has changed to "Show Finished" button and all the completed items are hidden from view. Clicking on the "Show Finished" button on this page will bring back the finished items to be shown below the unfinished items.

Before deleting all completed items

This is the page showing a to-do list with both uncompleted items and completed items. Next, we are going to delete all the completed items by clicking on the "Delete Finished" button.

After deleting all completed items

This is the page after the user clicked the "Delete Finished" button. The completed item, "Call Mom", has been removed from the to-do list.
